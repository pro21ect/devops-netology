```
2.  
Не правильный вариант скрипта
while ((1==1)
do
curl https://localhost:4757
if (($? != 0))
then
date >> curl.log
fi
done

Правильный вариант:
while ((1==1))
  do
  curl https://localhost:4757
  if (($? != 0))
  then
  date > curl.log
  break
  fi
  done
  Добавлена пропущенная скобка в 1-ой строке, команда break. Уменьшение используемого места
  на диске решается перезаписью лога.
```

3. Предлагаю такой вариант
#! /usr/bin/env bash
ip=("192.168.0.1" "173.194.222.113" "87.250.250.242") 
a=0
b=$((${#ip[@]}-1))
while [ $a -le $b ]
do
    echo доступность узла ${ip[$a]}
    for c in {1..5}
        do
           echo попытка достучаться $c из 5...
           curl -I -m 1 -sS http://${ip[$a]} &>> log
        done
let "a+=1"
done
