```
3. 
Принципы BASE и ACID в одной системе уживаться не могут.
ACID - данные согласованы, система надёжна
BASE - это производительность, данные могут быть не согласованны

```
4. 
Думаю речь идёт о Redis, она соответствует требованиям задачи
Минусы вытекают из архитектуры:
а) Большие требования к ОЗУ
б) Отсутствие SQL запросов
в) Сравнительно низкая надёжность
г) Базовый уровень безопасности, нет контроля достпа, делений на пользователей или группы.
д) Сравнительно худшая масшабиремость относительно реляционных БД
